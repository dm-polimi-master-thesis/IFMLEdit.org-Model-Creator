# IFMLEdit.org

A simple IFML Web Editor

## Installation

__To install all dependencies (and regenerate static assets)__
```bash
npm install
```

__To just regenerate static assets__
```bash
npm run-script build
```

## Development

__To start server__
```bash
npm start
```

__To stop server__
```bash
npm stop
```
